## 为什么选择 Xinbot？
- **彩色日志**：像官方客户端一样渲染颜色与格式。
- **正版登录**：可选的正版账号登录流程，更安心。
- **插件优先**：类 Bukkit 事件系统，快速扩展能力。
- **上手简单**：单 JAR、清晰配置、开箱即用。

---

## 🚀 快速开始

详细指南请访问我们的 **[官方文档网站](https://xinbot.shouldbe.top/zh/guide/getting-started)**。

1) **下载**: 从 [Releases](https://github.com/huangdihd/xinbot/releases) 获取最新版 JAR。
2) **配置**: 创建 `config.conf` (推荐使用 [在线配置生成器](https://xinbot.shouldbe.top/zh/guide/config-generator))。
3) **运行**:
   ```bash
   java -jar xinbot-[版本号].jar
   ```
