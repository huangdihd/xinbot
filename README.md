## Why Xinbot?
- **Vibrant Logs**: Parse and render messages just like the official client.
- **Secure Login**: Use your official Minecraft account with confidence.
- **Plugin-First**: Extend behavior with a familiar Bukkit-style event system.
- **Ready to Run**: One JAR, clear config, fast start.

---

## 🚀 Quick Start

For detailed guides, please visit our **[Official Documentation](https://xinbot.shouldbe.top/guide/getting-started)**.

1) **Download**: Get the latest JAR from [Releases](https://github.com/huangdihd/xinbot/releases).
2) **Config**: Create `config.conf` (use our [Config Generator](https://xinbot.shouldbe.top/guide/config-generator)).
3) **Run**:
   ```bash
   java -jar xinbot-[version].jar
   ```
